On Windows, get the latest versions of msysgit: http://code.google.com/p/msysgit/downloads/list and TortoiseGit: http://code.google.com/p/tortoisegit/wiki/Download. After installing, you should see a couple of additional items (“Git Bash”, “TortoiseGit ...”, ...) in the context menu when right-clicking on a folder in the Windows explorer.

On MacOS, download and install Git from here: http://git-scm.com/download/mac

Just use your package manager on Linux.
